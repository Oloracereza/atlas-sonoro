# Politica de privacidad

Atlas Sonoro es una app educativa para explorar subgeneros musicales.

## Datos que usa la app

- Correo electronico: se usa para crear cuenta e iniciar sesion con Firebase Auth.
- Favoritos: se guardan los generos que el usuario marca como favoritos.

La app no usa anuncios, no vende datos y no incluye contenido generado por usuarios.

## Servicios usados

La app usa Firebase Authentication y Cloud Firestore para login y favoritos.

## Eliminacion de datos

El usuario puede eliminar su cuenta desde la app en:

**Acerca de > Eliminar cuenta**

Al eliminar la cuenta tambien se eliminan los favoritos guardados.
